# EB Digital Studio — Portfolio v1

Efecan Berber için hazırlanmış premium, responsive ve statik portföy sitesi.

## Özellikler
- 6 kullanıcı seçilebilir renk teması
- Tema tercihini localStorage ile hatırlama
- Özel EB monogramı
- Ber Can’t Premium Online Coaching vaka çalışması
- Mobil menü ve tema paneli
- Scroll reveal animasyonları
- Telefon, e-posta ve WhatsApp iletişim bağlantıları
- GitHub Pages uyumlu yapı

## Yayınlama
1. Bu klasörün içindeki dosyaları GitHub reposunun kök dizinine yükleyin.
2. Settings → Pages → Deploy from a branch → main / root seçin.
3. Alan adı aldığınızda `index.html` içindeki Open Graph adreslerini ve bir `CNAME` dosyasını güncelleyin.
