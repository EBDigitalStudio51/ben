# EB Digital Studio — Signature Edition

Efecan Berber için hazırlanan premium, tek sayfalık portföy sitesi.

## Özellikler
- 6 kullanıcı seçilebilir renk teması
- Sinematik açılış
- Etkileşimli parçacık arka planı
- Özel imleç, 3D tilt ve manyetik butonlar
- Ber Can’t vaka çalışması
- Hizmetler, kişisel hikâye, teknoloji yörüngesi ve çalışma süreci
- WhatsApp teklif oluşturucu
- Mobil uyumluluk, azaltılmış hareket desteği ve özel 404

## Yayın
Dosyaları GitHub reposunun köküne yükleyin. Settings → Pages → Deploy from a branch → main → /(root).

## Alan adı ayarı
Alan adı alındığında `robots.txt`, `sitemap.xml` ve gerekirse Open Graph URL alanlarını gerçek alan adına göre güncelleyin.
