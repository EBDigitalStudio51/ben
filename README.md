# EB Digital Studio — Experience OS 3.0

Efecan Berber için hazırlanan interaktif premium portföy ve dijital yetenek vitrini.

## Yeni Experience OS özellikleri

- **Canlı Design Lab:** sektör, tasarım dili, renk atmosferi ve sistem özellikleri seçilerek anlık site konsepti üretir.
- **EB Command Center:** `Ctrl + K` ile proje, tema, vaka çalışması, iletişim ve teklif bölümlerine hızlı erişim.
- **Ber Can’t tam vaka çalışması:** kullanıcı tarafından sağlanan gerçek eski yapı, yeni sistemle sürüklenebilir önce/sonra alanında karşılaştırılır.
- **Gerçek eski sürüm ön izlemesi:** `legacy/bercant-old/` içinde yalnızca görsel sunum amacıyla güvenli ve etkileşimsiz arşiv.
- **Journey Log:** Efecan Berber’in eğitimden ilk projeye ve ajans hedefine uzanan yolculuğu.
- **Tema atmosferleri:** altı temada yalnızca renkler değil, arka plan dokusu ve tasarım atmosferi de değişir.
- **Teklif entegrasyonu:** Design Lab seçimleri proje sihirbazına aktarılır ve WhatsApp mesajına dönüşür.
- Sinematik intro, parçacık sistemi, özel imleç, tilt kartlar, mobil uyumluluk ve azaltılmış hareket desteği korunur.

## Yayın

ZIP içeriğini GitHub reposunun köküne yükleyin:

`Settings → Pages → Deploy from a branch → main → /(root)`

Mevcut dosyaların üzerine yazılmalıdır. `case-study-bercant.html`, `css/experience.css`, `js/experience.js` ve `legacy/` klasörü mutlaka yüklenmelidir.
